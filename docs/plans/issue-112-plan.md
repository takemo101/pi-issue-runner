# Issue #112 実装計画

## 概要

`agents/plan.md` に「ディレクトリがなければ作成」の指示を追加する。
AIが計画書を保存する際、`docs/plans/` ディレクトリが存在しない場合でも正しく動作するようにする。

## 影響範囲

- `agents/plan.md` - 唯一の変更対象ファイル

## 実装ステップ

1. `agents/plan.md` の「### 3. 出力」セクションを特定
2. 現在の指示文の前に、ディレクトリ作成の指示を追加

## 変更内容

**Before:**
```markdown
### 3. 出力
計画書は `docs/plans/issue-{{issue_number}}-plan.md` に保存してください。
```

**After:**
```markdown
### 3. 出力
ディレクトリがない場合は作成してから、計画書を `docs/plans/issue-{{issue_number}}-plan.md` に保存してください。
```

## テスト方針

- ファイルの構文が正しいことを確認（マークダウン形式）
- 指示文が明確で理解しやすいことを確認

## リスクと対策

- **リスク**: なし（単純なテキスト追加のみ）
- **対策**: 不要
