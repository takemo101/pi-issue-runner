# Memos

開発中のメモ、検討資料、議事録などを格納するディレクトリです。

## 目的

- 開発に関するメモ・備忘録
- 設計検討の途中経過
- 議事録・ミーティングノート
- アイデアやブレインストーミングの記録

## `docs/plans/` との違い

| ディレクトリ | 目的 | 内容 |
|-------------|------|------|
| `docs/plans/` | Issue実装計画 | 特定のIssueに対する正式な実装計画書 |
| `docs/memos/` | 非公式なメモ | 検討中のアイデア、議事録、備忘録など |

## ファイル命名規則（推奨）

- 日付プレフィックス: `YYYY-MM-DD-topic.md`
- 例: `2026-01-31-performance-ideas.md`

## 注意

- このディレクトリ内のドキュメントは暫定的なものです
- 正式な設計ドキュメントは `docs/` 直下に配置してください
